## 개발 이야기

지금껏 거론 된적 없는 아이디어 조각들을 모아 하나의 서비스로 만드는 과정을 직접 경험해보는것도 색다른 경험이었습니다.

기획부터 디자인, 프론트엔드까지 모두 할수 있었거든요.

디자인은 다소 급조된 부분이 있어 아쉽게 생각됩니다.

차후 Figma로 다시 만들어보면서 디자인 다듬어 볼 생각입니다.

프론트엔드는 리액트를 사용했습니다.

스타일은 SASS를 사용했고, 절대경로 처리를 위해 craco를 썼습니다.
